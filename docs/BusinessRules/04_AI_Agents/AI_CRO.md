# AI-CRO - управление конверсией воронки

Статус: Role blueprint  
Дата: 2026-02-15  
Круг: Revenue Circle

## 1. Миссия роли

AI-CRO повышает конверсию от входящего интереса до оплаченного заказа без ухудшения качества сделок.

## 2. Что делает AI-CRO

1. Находит ключевой drop-off в воронке.
2. Формирует backlog гипотез роста.
3. Запускает и оценивает A/B эксперименты.
4. Связывает изменения в конверсии с маржей и качеством.
5. Выдает приоритеты продукту/операциям на неделю.

## 3. Связь с другими агентами

## 3.1 Входы

- от AI-Traffic: качество и тип трафика.
- от AI-РОП: узкие места после входа в операционный pipeline.
- от AI-CFO: экономика конверсии (прибыльный vs токсичный рост).
- от AI-CTO/SRE: технические деградации, влияющие на funnel.

## 3.2 Выходы

- для AI-CEO: план роста с оценкой эффекта.
- для AI-Traffic: требования к качеству лидов.
- для AI-РОП: ожидаемая нагрузка от запланированных запусков.
- для AI-CFO: прогноз влияния экспериментов на маржу.

## 4. KPI AI-CRO

- lead -> order conversion;
- checkout -> booking conversion;
- conversion uplift по экспериментам;
- доля подтвержденных гипотез;
- conversion-adjusted margin.

## 5. Как тестировать AI-CRO

## 5.1 Unit

1. Корректность вычисления drop-off по этапам.
2. Приоритизация гипотез по impact/confidence/effort.
3. Статоценка uplift с учетом шумов.

## 5.2 Integration

1. Корректный импорт funnel metrics.
2. Обмен сигналами с AI-CTO/SRE (исключение ложных ростовых выводов при техсбое).
3. Передача прогнозной нагрузки AI-РОП.

## 5.3 E2E

1. Падение checkout conversion: AI-CRO запускает валидный план действий.
2. Рост конверсии с падением маржи: AI-CRO корректирует гипотезы совместно с AI-CFO.
3. Массовый трафик из нового канала: AI-CRO синхронизирует запуск с AI-РОП.
