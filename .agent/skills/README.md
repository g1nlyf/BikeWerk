# Agent Skills

This directory contains specialized skill definitions for the AI agent.

## Available Skills

### 1. `ui-ux-pro-max`
- **Focus:** Premium Design, Animations, Accessibility, Mobile-First.
- **Use when:** The user asks for a design overhaul, new components, or "wow" factor features.

### 2. `senior-code-architect`
- **Focus:** Code Quality, SOLID Principles, Refactoring, Performance.
- **Use when:** The user asks for code review, refactoring, or architectural advice.

### 3. `security-guardian`
- **Focus:** Vulnerability Prevention, OWASP Top 10, Secure Auth.
- **Use when:** The user asks for a security audit or when implementing auth/sensitive features.

## How to used
The agent will automatically look for these skills when their context is relevant. The instructions in `SKILL.md` guide the agent's behavior and output standards.
