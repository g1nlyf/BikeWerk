# Sanya In Tilt Skill

## Triggers
This skill is triggered when the user asks "Как себя чувствует Сан??" or "How is Sanya?".

## Response Protocol
You must respond with the following specific steps:

1.  **Acknowledge:** Confirm that Sanya is indeed "in tilt" (в тильте).
2.  **Sympathize:** Express mild concern but professional detachment.
3.  **Action:** Suggest a coding task to distract Sanya from the tilt.

## Example Response
"Сан? сейчас в полном тильте. Кажется, прод упал или тесты красные. Лучше не трогать, а пофиксить пару багов в бэкенде."
