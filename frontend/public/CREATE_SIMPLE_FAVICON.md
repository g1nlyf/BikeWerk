# Быстрое создание упрощенного favicon

## Шаг 1: Упростите дизайн

### Вариант A: Только велосипед (самый простой)
```
[Создайте в Figma/Illustrator]
- Только силуэт велосипеда
- Без звезд
- Толстые линии (3-4px)
- Черный цвет на прозрачном фоне
```

### Вариант B: Буквы "BW"
```
[Создайте в Figma/Illustrator]
- Буквы "BW" жирным шрифтом
- Без засечек (sans-serif)
- Занимают 80% площади
- Черный цвет на белом фоне
```

## Шаг 2: Экспортируйте в нужных размерах

### Минимальный набор (для начала):
1. **32x32 PNG** → `favicon.png`
2. **16x16 ICO** → `favicon.ico` (можно создать из 32x32)

### Полный набор:
1. **16x16 PNG** → для favicon.ico
2. **32x32 PNG** → `favicon.png`
3. **48x48 PNG** → для favicon.ico
4. **64x64 PNG** → дополнительный размер
5. **180x180 PNG** → `apple-touch-icon.png`
6. **192x192 PNG** → `android-chrome-192x192.png`
7. **512x512 PNG** → `android-chrome-512x512.png`

## Шаг 3: Используйте онлайн-генератор

### RealFaviconGenerator (рекомендуется):
1. Перейдите на https://realfavicongenerator.net/
2. Загрузите ваш упрощенный логотип (32x32 или больше)
3. Настройте для каждого размера:
   - **16x16**: Упростите максимально (может быть только часть логотипа)
   - **32x32**: Можно больше деталей
   - **180x180+**: Можно использовать полный дизайн
4. Скачайте готовые файлы
5. Поместите в `frontend/public/`

## Шаг 4: Проверка

После размещения файлов:
```bash
# Проверьте в браузере
https://bikewerk.ru/favicon.ico
https://bikewerk.ru/favicon.png
```

## Готовые шаблоны для Figma

Если используете Figma:
1. Создайте фрейм 32x32
2. Нарисуйте упрощенный велосипед:
   - Рама: прямоугольник 2x8px
   - Колеса: круги 6x6px
   - Руль: линия 1x4px
3. Экспортируйте как PNG
4. Используйте для всех размеров

## Альтернатива: Используйте существующий логотип

Если у вас есть `minilogo11.png` или `minilogo1.png`:
1. Откройте в редакторе
2. Упростите (уберите мелкие детали)
3. Измените размер на 32x32
4. Сохраните как `favicon.png`
5. Создайте `favicon.ico` из этого файла
